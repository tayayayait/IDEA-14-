# Step 4 HS 854232 미국 상세조회 검증

## 확인 조건
- 국가: US / 미합중국
- 제품: 반도체(DRAM)
- HS: 854232
- HSK: 8542321010

## 직접 조회 결과
- KOTRA 수입규제품목 API `DS00000128`: 총 17,895건 조회 기준 `HSCD`가 `854232`로 시작하거나 `8542321010`과 일치하는 행 0건.
- KOTRA 국별 대세계 수입규제 CSV `20250603`: 총 4,942행 기준 HS 코드 컬럼 1~128에서 `854232` 또는 `8542321010` 매칭 0건.
- KOTRA 해외인증 API `overseasAuthInfo`: `search5=854232` 조건에서 미국 인증 1건 반환.

## 확인된 이슈
- 해외인증 API는 국가명을 `미국`으로 반환한다.
- 앱의 US alias에는 기존에 `미국`, `미합중국`이 없어 `country_name=미합중국(The United States of America)`와 API row `nat=미국`을 같은 국가로 판정하지 못했다.
- 그 결과 `hs_only:1` 응답이 있었지만 최종 인증 목록에는 0건으로 표시됐다.

## 수정
- `country-detail`의 `COUNTRY_ALIASES`에 KOTRA가 반환하는 한국어 국가명을 추가했다.
- US alias에 `미국`, `미합중국`을 추가하여 HS 854232 미국 인증 row가 최종 매칭될 수 있게 했다.
