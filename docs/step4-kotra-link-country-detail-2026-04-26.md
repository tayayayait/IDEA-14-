# Step4 KOTRA 규제 링크 보정 (국가 상세 우선)

Date: 2026-04-26

## 문제
- `규제·NTM`의 `KOTRA` 링크가 `MENU_ID=3700` 초기 화면으로 열리는 케이스가 남아 있었다.
- 원인: 링크 생성 시 `row.raw.country_code_iso2`를 우선 사용했는데, 일부 데이터에서 `ISO_WD2_NAT_CD`가 현재 선택 국가와 다르게 내려와 매핑 실패 또는 오매핑이 발생했다.

## 적용
- 규제 링크 생성 함수 `buildKotraRegDetailUrl`에 옵션 `countryCodeIso2`를 추가했다.
- Step4 렌더링에서 `cc`(현재 선택 국가)를 우선 전달하도록 변경했다.
- 결과적으로 KOTRA 링크는 가능한 경우 항상 `pRegnCd/pNatCd`가 포함된 국가 상세 진입 URL을 생성한다.

## 참고
- 본 변경은 “국가 기준 상세 진입” 보정이다.
- KOTRA 외부 사이트의 “개별 규제 항목 1건 상세 딥링크”는 공개 GET 규칙이 확인되지 않아 여전히 지원하지 않는다.
