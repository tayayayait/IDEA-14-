# Step2 제품 설명 문단형 특징 추가 (2026-04-28)

## 원인
- 기존 `ai-product-description` 출력은 라벨형 템플릿(`개요`, `특징`, `프로젝트 맥락`)에 가까워 사용자가 원하는 일반 설명문 형태가 아니었다.
- 예: `반도체(DRAM)` 입력 시 “반도체란 무엇인가요?”로 시작해 정의와 전기적 특징을 자연문장으로 설명해야 한다.

## 변경
- 제품 설명 최종 출력에서 `개요:`, `특징:`, `프로젝트 맥락:` 라벨을 제거했다.
- AI 프롬프트에 “한 문단의 자연문장”과 “`제품명`란 무엇인가요?” 형식의 정의 문장을 요구했다.
- AI 응답의 `overview`, `features`, `project_context` 필드는 내부 합성용으로만 사용하고, 최종 설명은 자연문장으로 합친다.
- AI 실패 시에도 규칙 기반 폴백에서 제품 정의, 특징/원리, 프로젝트 정보를 한 문단으로 생성한다.
- `축전지 제조`처럼 업종형 접미사가 붙은 입력은 설명 주체를 `축전지`로 정리한다.

## 예시
```text
반도체(DRAM)란 무엇인가요? 반도체(DRAM)는 데이터를 일시적으로 저장하고 빠르게 읽고 쓰기 위해 사용하는 메모리 반도체입니다. 전기 신호의 저장 상태를 이용해 데이터 비트를 표현하며, 반복적인 읽기·쓰기 작업과 전자기기의 메모리 기능이 핵심입니다. 본 프로젝트는 회사 삼성전자(주), 지역 경기도, 산단 화성일반산업단지, 산업분류코드 26111 정보를 기반으로 검토합니다.
```

## 배포
- 운영 반영에는 `ai-product-description` Edge Function 재배포가 필요하다.
