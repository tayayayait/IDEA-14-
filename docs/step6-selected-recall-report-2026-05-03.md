# 6단계 리포트 리콜 선택값 반영 수정

## 배경

5단계 `KC인증·리콜정보`에서 사용자가 국내 리콜과 국외 리콜 후보를 체크해도, 6단계 리포트의 리콜 상세 영역이 선택 원본이 아닌 `project_safety_flags`의 리콜 요약 행을 기준으로 표시되는 문제가 있었다.

## 변경 사항

- 5단계에서 리포트로 이동할 때 현재 체크박스 선택값을 `product_safety.raw.selected_*_ids`에 먼저 저장한다.
- 6단계 `KC인증·리콜정보` 영역은 `product_safety.raw.domestic_recalls`와 `product_safety.raw.foreign_recalls`를 선택 ID로 필터링해 표시한다.
- 국내 리콜과 국외 리콜을 별도 섹션으로 표시한다.
- 각 리콜 항목에는 국내/국외 구분, 제품명 또는 모델명, 공표일, 제품결함/위해내용, 소비자 행동요령/조치사항을 표시한다.
- AI 리포트 생성용 근거 데이터에서도 선택되지 않은 리콜 요약 행을 제외하고, 선택된 리콜 상세만 전달한다.

## 유지 사항

- 기존 KC 인증 선택 및 상세 표시 로직은 유지한다.
- 선택 ID가 없는 기존 데이터는 기존 요약 플래그 기반 근거 생성을 유지한다.
