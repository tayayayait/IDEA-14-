# Step1 Pagination Update (2026-04-23)

## 변경 목적

- `Step1Company`의 검색 결과를 한 페이지에 10건씩 표시하도록 제한
- 하단 페이지 번호 버튼으로 전체 결과를 탐색하도록 개선

## 변경 파일

- `src/pages/Step1Company.tsx`

## 상세 변경

1. 페이지 상태 추가
- `currentPage` 상태와 `PAGE_SIZE = 10` 상수 추가
- 검색 실행/조건 초기화 시 `currentPage`를 1로 리셋
- 결과 길이 변화 시 현재 페이지가 유효 범위를 벗어나면 자동 보정

2. 렌더링 데이터 변경
- `results` 전체 렌더링에서 `pagedResults` 렌더링으로 전환
- 모바일 카드 목록(`MobileCardList`)과 데스크톱 테이블 모두 동일한 페이지 슬라이스 사용

3. 페이지네이션 UI 추가
- 결과 영역 하단에 `이전`, 페이지 번호, `다음` 버튼 추가
- 현재 페이지/총 페이지와 총 결과 건수 표시

## 기대 효과

- 대량 조회 시 한 화면에 과도한 행이 노출되는 문제 완화
- 사용자가 하단 페이지 번호를 통해 전체 조회 결과를 순차 탐색 가능
