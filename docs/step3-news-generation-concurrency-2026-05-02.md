# Step3 상위 3개국 뉴스 생성 속도 개선

## 변경 목적

상위 3개국 뉴스 근거 생성은 국가별 품질 기준을 유지하되, 순차 실행으로 인한 대기 시간을 줄인다.

## 변경 내용

- 상위 3개국 `recommend-country-news` 호출을 순차 처리에서 동시성 2개 배치 처리로 변경했다.
- 한 국가 처리가 끝날 때마다 Step3 국가 데이터를 즉시 다시 읽어 화면에 반영한다.
- 뉴스 결과 갱신 중에는 `api-customs-trade` 재조회가 반복되지 않도록 `load({ refreshCustoms: false })` 경로를 추가했다.
- 검색어 수, KOTRA 검색 기준, AI 판정 기준, 뉴스 품질 필터는 변경하지 않았다.

## 기대 효과

- 국가 3개를 `1 -> 2 -> 3` 순서로 기다리는 구조보다 체감 시간이 줄어든다.
- 동시성은 2개로 제한해 외부 API 부하와 실패 가능성을 낮춘다.
- 품질 기준은 기존 `news_scope`, 제품/국가 관련성, AI/규칙 판정 흐름을 그대로 사용한다.
