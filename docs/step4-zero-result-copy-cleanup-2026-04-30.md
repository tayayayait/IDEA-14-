# Step4 0건 안내 문구 정리 (2026-04-30)

## 변경 목적
- 인증/규제 상세 결과가 0건일 때 사용자 화면에 검색조건, API 응답 상태, 캐시 조회 시도, 대체 확인 링크 등 내부 진단 정보를 노출하지 않는다.
- 0건은 검토 요청이 아니라 결과 부재로 표시한다.

## 변경 내용
- `src/pages/Step4CountryDetail.tsx`
  - `DetailZeroResultNotice`에서 0건 상태는 제목 문구만 표시하도록 수정.
  - `사용 검색조건`, `판정 상태`, `API 응답 상태`, `출처 유형`, `검색 시도 요약`, `대체 확인 링크`, `기관 확인 필요` 문구를 0건 안내에서 제거.
  - 오류/오래된 캐시 상태는 한 줄 재실행 안내만 유지.
