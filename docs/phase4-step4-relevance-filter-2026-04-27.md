# Phase 4 - Step 4 품목·국가 관련성 필터 강화 (2026-04-27)

## 목적
- Step 4 인증/규제 결과에서 품목-국가 무관 항목(예: 차량부품 분석 시 식품/임플란트/비데 등) 노출을 차단한다.

## 변경 파일
- `supabase/functions/country-detail/index.ts`

## 핵심 변경
1. 인증 데이터(KOTRA 해외인증정보) 필터 강화
- 기존: 국가 매칭 실패 시 비필터 원본으로 fallback 가능.
- 변경: 국가 필터 이후 `HS 또는 제품 토큰` 관련성이 있는 항목만 통과.
- 관련성 항목이 없을 때 비필터 fallback 제거, 국가 필터 결과 내에서만 fallback.

2. 수입규제(KOTRA 수입규제정보) 랭킹/필터 강화
- 기존: 점수 > 0이면 포함되어 국가 신호만으로 무관 품목이 포함될 수 있었음.
- 변경: `국가 신호` + (`HS 매칭` 또는 `제품 토큰 매칭`)을 만족하는 항목만 포함.
- 관련성 조건을 만족하지 못하면 결과에서 제외.

3. 제품 토큰 전처리 강화
- `부품/제품/소재` 등 일반 토큰을 약한 토큰으로 분류해 관련성 오탐을 줄임.
- 약한 토큰만 남는 경우를 대비해 원본 토큰 fallback 유지.

## 기대 효과
- Step 4 인증/규제 목록이 입력 품목 및 대상 국가와 더 직접적으로 연결됨.
- 공모전 심사 관점에서 근거 데이터 신뢰성 개선.

## 비고
- SafetyKorea는 API 승인 대기 정책을 유지하며 이번 변경 범위에 포함하지 않음.
