# Step3 제품 일반형 뉴스 근거 품질 기준

## 변경 목적

Step3 상위 3개국 뉴스 근거는 특정 품목에 하드코딩하지 않고 모든 HS/HSK 품목에 동일하게 적용되는 기준으로 선별한다. 직접 뉴스가 없을 때 무관한 뉴스를 채우지 않으며, 선택 국가 뉴스와 타국 공급망 참고뉴스를 분리한다.

## 적용 기준

- 직접 뉴스는 선택 국가와 제품/HS 직접 근거가 모두 확인될 때만 `selected_country_direct`로 저장한다.
- 선택 국가의 관세, 제재, 환율, GDP, 경기, 물류, 통관 등은 `selected_country_export_env`로 저장한다.
- 선택 국가의 제품군, 수요처, 공급망 연결이 방어 가능한 산업 뉴스는 `selected_country_industry`로 저장한다.
- 선택 국가가 아닌 국가 뉴스가 공급망이나 경쟁국 맥락에서만 유용하면 `supply_chain_reference`로 분리한다.
- 5년 초과 뉴스는 `archive_reference`로만 표시한다.

## 품질 보정

- `data`, `access`, `physical`, `에서`, `주요`, `위해`, `관련`, `데이터`, `접근`, `물리적` 등 단독 의미가 약한 토큰은 직접성 근거로 사용하지 않는다.
- 제품명에 포함된 일반어는 제거하지 않고 약한 토큰으로 강등한다.
- `snack`, `potato chips`, `간식` 문맥의 `chip`은 전자부품/반도체 제품 근거로 인정하지 않는다.
- `fabric`, `textile`, `섬유`, `의류`, `garment` 문맥은 해당 제품군이 아닌 경우 제품 직접 근거로 인정하지 않는다.

## UI 표시

뉴스 근거 섹션은 다음 순서로 표시한다.

1. 선택 국가 직접 뉴스
2. 선택국 수출환경 뉴스
3. 선택국 산업 배경뉴스
4. 공급망/경쟁국 참고뉴스
5. 과거 참고 근거

동일한 `산업/수출환경 배경뉴스` 제목을 반복하지 않고, `news_scope` 기준으로 각 근거의 역할을 명확히 표시한다.
