# Step 3 한국어 UI 문구 정리

## 변경 범위
- Step 3 후보국 추천 화면의 뉴스 근거 카드 제목, 설명, 선택 국가 드롭다운, 원문 링크, 최신성 배지, 과거 참고 근거 라벨을 한국어로 변경했다.
- 전체 후보국 점수표의 제목, 설명, 컬럼명, 상세 버튼, 접근성 라벨을 한국어로 변경했다.
- 추천 실행 중 진행 상태 메시지를 한국어로 변경했다.
- 기존 저장 데이터에 포함된 `recent<=1y`, `category:*`, `[Export impact]` 형식의 뉴스 근거 표시 문구를 화면 표시 시 한국어로 정규화했다.
- `[수출 영향] 경기, 달러, 금리 can affect DRAM, NAND Flash in cost, lead-time, or payment conditions.` 형식의 저장 문구는 `[수출 영향] 경기, 달러, 금리 변동은 DRAM·NAND Flash의 원가, 납기, 결제 조건에 영향을 줄 수 있습니다.` 형식으로 정규화한다.
- 뉴스 근거/리포트 표시 문자열의 `&hellip;` HTML 엔티티는 화면에 그대로 노출하지 않고 `...`로 정규화한다.
