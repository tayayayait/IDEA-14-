# Step 3 상위 3개국 뉴스 근거 3순위 빈 결과 보강 (2026-05-02)

## 원인

- `recommend-country-news`는 먼저 직접/배경 뉴스 후보를 엄격한 제품 수출 적합성 기준으로 선별한다.
- 3순위 국가에서 KOTRA 뉴스 조회는 성공해도, 엄격 필터를 통과한 후보가 없으면 `sources=[]`가 저장된다.
- 이 경우 버튼은 실행됐지만 Step 3 UI에는 "선택한 국가에 분류된 뉴스 근거가 없습니다."로 표시된다.

## 수정

- 엄격 필터 결과가 0건일 때 KOTRA 국가 뉴스 조회 결과 중 선택국가와 매칭되는 최신 후보를 `country_background`로 저장한다.
- 이 보강 결과는 `score_relevant=false`로 저장해 직접 제품 뉴스나 점수 반영 근거로 승격하지 않는다.
- 직접 뉴스 없음 플레이스홀더는 계속 저장하지 않는다.

## 검증

- 상위 3개국 뉴스 근거 생성 시 3순위 국가도 직접 근거가 없으면 배경 뉴스 후보를 표시할 수 있다.
- 직접 뉴스 영역에는 실제 직접 뉴스만 표시된다.
