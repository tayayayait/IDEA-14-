# Step3 뉴스 근거 표시 한도 분리 (2026-05-03)

## 변경 목적

뉴스 근거 패널에서 선택국 수출환경 뉴스와 선택국 산업 배경뉴스가 하나의 4건 제한을 공유해 한쪽 카테고리가 다른 카테고리를 밀어내는 문제를 수정했다.

## 변경 내용

- 제품 직접 뉴스는 기존처럼 최대 4건 표시한다.
- 선택국 수출환경 뉴스는 최대 4건 표시한다.
- 선택국 산업 배경뉴스는 최대 4건 표시한다.
- 공급망/경쟁국 참고뉴스는 기존처럼 최대 4건 표시한다.
- 저장된 뉴스 근거 schema와 Supabase Edge Function 저장 로직은 변경하지 않았다.

## 제품 직접뉴스가 비어 있는 경우

제품 직접뉴스 섹션은 `news_scope = selected_country_direct`인 항목만 표시한다.
이 값은 대상국 직접 매칭과 AI의 `direct_product` 판정 또는 규칙상 직접 제품 신호가 함께 잡힌 후보에만 붙는다.
따라서 화면에 수출환경/산업 배경뉴스만 보이면, 해당 실행에서 제품명·HS 코드 기준 직접 뉴스로 저장된 후보가 없었다는 뜻이다.

## 검증

- `src/test/step3-target-market-insight.test.ts`에 수출환경 4건과 산업 배경 4건이 별도로 표시되는 회귀 테스트를 추가했다.
