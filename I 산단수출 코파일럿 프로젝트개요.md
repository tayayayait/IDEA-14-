## 프로젝트 배경 및 목표

산업단지 제조기업은 수출을 준비할 때 여러 기관의 데이터를 따로 확인해야 한다. 공장·생산품 정보는 한국산업단지공단, 국가·시장·인증·수입규제 정보는 KOTRA, 국가·업종·결제위험은 한국무역보험공사, 전략물자 관련 정보는 무역안보관리원, 제품안전·리콜 정보는 SafetyKorea 계열에서 각각 확인해야 한다.

이 구조는 수출 준비 시간을 늘리고, 인증 누락, 수입규제 미확인, 결제위험 과소평가, 전략물자 검토 누락, 제품안전 이슈 미확인 같은 실무 리스크를 만든다.

AI 산단수출 코파일럿의 목표는 공공데이터를 단순 조회하는 것이 아니라, 산업단지 제조기업의 수출 준비 과정을 실제 실행 가능한 의사결정 흐름으로 바꾸는 것이다.

주요 목표:

- 기업·공장·생산품 정보를 기반으로 수출 준비 대상을 정의한다.
- 제품명과 HS/HSK 후보를 기준으로 수출 후보국을 추천한다.
- 국가별 해외인증, 수입규제, 시장동향을 비교한다.
- 국가·업종·결제 리스크를 수출국 추천 점수에 반영한다.
- 전략물자 여부는 자동 판정하지 않고 검토대상 플래그로만 제공한다.
- 제품안전·리콜 유사사례를 확인해 사전 리스크를 줄인다.
- 모든 AI 요약 결과에 원천 데이터와 근거를 표시한다.
- 최종적으로 1페이지 수출 실행 리포트를 생성한다.

### 1. 기업·공장 검색

| 항목 | 내용 |
|---|---|
| 기능 설명 | 회사명, 공장명, 지역, 생산품 키워드로 기업·공장 정보를 조회한다. |

| 출력값 | 회사명, 공장주소, 업종, 생산품, 공장 기본정보 |
| 활용 API | 한국산업단지공단_공장등록생산정보조회서비스 |

## 비기능 요구 사항

| 항목 | 요구 사항 |
|---|---|
| 데이터 신뢰성 | API별 제공기관, 조회일, 원문 링크를 표시한다. |
| AI 답변 제한 | 근거 데이터가 없으면 `확실한 정보 없음`으로 표시한다. |
| 법적 리스크 | 인증, 수입규제, 전략물자 정보는 참고용이며 최종 판단은 관련 기관 확인이 필요하다고 명시한다. |
| 개인정보·영업비밀 | 제품 스펙, 거래국가, 내부 메모는 비공개 저장한다. |
| 라이선스 | KOTRA 해외시장뉴스 등 재배포 제한 가능성이 있는 데이터는 요약·메타데이터·원문 링크 중심으로 사용한다. |
| 성능 | 대표 시나리오 기준 분석 결과는 10초 이내 1차 표시를 목표로 한다. |
| 확장성 | MVP에서는 P0 API 10개만 사용하고, 추가 기관 데이터는 사용하지 않는다. |
